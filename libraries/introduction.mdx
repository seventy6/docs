## What are SDKs and Libraries?

The Auth API provides SDKs and Libraries to help you get started quickly. Our SDKs and Libraries are open-source and available on GitHub. They are designed to help you interact with the Auth API in a more natural way.

### SDKs

<Card title="NPM Client" icon="link" href="npm-package">
  The Auth API NPM client is a Node.js client for the The Auth API. It provides
